公司介绍

茅台集团保健酒业有限公司隶属于茅台酒厂股份有限责任公司，公司始终坚守效益第一的道路， 经年坚守古法纯粮醇酿把持着白酒防线，工匠般坚持着出酒的配比，五斤高粱一斤酒的严谨态度， 秉持着“好酒知心”的匠人心态，保证系列产品酒质稳定，是“中国企业形象AAA企业”。 公司于2002年就通过企业ISO9001，和2008年质量体系认证。在秉持酱香白酒的同时， 更兼容了浓香白酒所赋予的芳香浓郁、绵柔甘冽、入口甜、落口绵、尾净余长特点扶正浓香白酒的古法工艺， 让时间修炼浓香的醇度，静藏磨练浓香的厚度。在多方精英会同下推出“䣽”酒品牌。“䣽”者醍醐灌顶是也， 古字今用寓意深远。让白酒成为文化的寄语，让文化定义生活，文化渗入生活。抚元白酒，走出浮躁和扭曲。并由酱印酒业有限公司运营全国市场， 成立茅台集团“䣽”酒全国营销中心，总部设在仁怀市坛厂工业园区。

贵州茅台酒厂(集团)保健酒业有限公司在全国设立“知酒堂”生活体验馆， 以全新的市场营销配给模式形成网络全覆盖，与各层次消费者建立良好的渠道，携手共建平稳、 健康的白酒文化和夯实事业的社会资源共享环境。
